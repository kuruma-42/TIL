# Recursion
- 재귀함수(Recursion)는 정의 단계에서 자신을 재참조하는 함수
- 전달되는 상태인 매개변수가 달라질 뿐 똑같은 일을 하는 함수
- 큰 문제를 작은 부분 문제로 변경하여 해결할 때 사용 

### Caution 
- 반드시 기저사항을 써야한다. 
- 사이클이 있다면 쓰면 안 된다 ( f(a) -> <- f(b)) 
- 반복문으로 될 것 같으면 반복문으로. (함수 호출에 대한 코스트) 

### Example
- 팩토리얼 n! : 그 이전의 항을 모두 곱하는 형태 
- 피보나치 : 0, 1, 1, 2, … 

```swift
func fact(n: int) -> Int {
  if( n == 1 | n == 0) {
    return n		
  } 
    return n * fact(n - 1)
} 
```
